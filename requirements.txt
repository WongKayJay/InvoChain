## SYSTEM REQUIREMENTS

### Development Environment

Download: https://flutter.dev/docs/get-started/install
Verify: flutter --version

#### 3. Git
Version: 2.x or higher
Download: https://git-scm.com/
Verify: git --version

#### 4. Code Editor (Choose one)
- Visual Studio Code (recommended) - https://code.visualstudio.com/
- Android Studio
- IntelliJ IDEA
- Any text editor

## ══════════════════════════════════════════════════════════════
## QUICK START INSTALLATION GUIDE
## ══════════════════════════════════════════════════════════════

### Step 1: Clone Repository
```bash
git clone https://github.com/WongKayJay/InvoChain.git
cd InvoChain
```

### Step 2: Install Backend Dependencies
```bash
cd backend
npm install
cd ..
```

### Step 3: Install Website Dependencies
```bash
cd frontend/website
npm install
cd ../..
```

### Step 4: Install Flutter Dependencies
```bash
cd frontend/mobile-app
flutter pub get
cd ../..
```

### Step 5: Configure Environment
```bash
# Create backend/.env file
cd backend
echo "PORT=3000" > .env
echo "NODE_ENV=development" >> .env
echo "JWT_SECRET=your_secret_key_here" >> .env
echo "DB_PATH=./database/invochain.db" >> .env
cd ..
```

### Step 6: Run Development Servers
```bash
# Terminal 1: Backend
cd backend
npm start

# Terminal 2: Website
cd frontend/website
npm run dev

# Terminal 3: Flutter App
cd frontend/mobile-app
flutter build web
python -m http.server 8080 -d build/web
```

Or use PowerShell commands:
```powershell
invochain-all    # Starts all services in separate windows
```

## ══════════════════════════════════════════════════════════════
## TROUBLESHOOTING
## ══════════════════════════════════════════════════════════════

### Issue: npm install fails
Solution: Clear npm cache
```bash
npm cache clean --force
npm install
```

### Issue: Flutter build fails
Solution: Upgrade Flutter and clean build
```bash
flutter upgrade
flutter clean
flutter pub get
flutter build web
```

### Issue: Port already in use
Solution: Kill process or use different port
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Linux/Mac
lsof -i :3000
kill -9 <PID>
```

### Issue: Permission denied
Solution: Run with appropriate permissions
```bash
# Linux/Mac
sudo npm install
sudo chown -R $(whoami) ~/.npm

# Windows
# Run terminal as Administrator
```
